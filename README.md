# 👁️ Gardly 👁️

**A family finance app for parents to manage their kids' allowances, balances, and cards all in one place.** 💳

> 🤖 This project was written entirely by AI — not a single line of code was touched manually by a human.

## ✨ How it was built

The design workflow for Gardly went like this:

1. **Designs were generated with [Google Stitch](https://stitch.withgoogle.com/)** — an AI-first design tool that turns ideas into UI.
2. **Designs were exported to [Figma](https://figma.com/)** for refinement and handoff.
3. **Code was generated from Figma** using Cursor's Figma integration, producing production-ready React components with pixel-perfect fidelity.

## 🚀 Running locally

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

Opens [http://localhost:3000](http://localhost:3000) in your browser. The page reloads automatically as you make changes.

### Run tests

```bash
npm test
```

Launches the test runner in interactive watch mode.

### Build for production

```bash
npm run build
```

Bundles the app for production into the `build` folder, optimised and minified.
